Noninformative priors (Berger $$3.3)

Why noninformative priors? (Berger $$3.3.1)

In many decision problems there is no natural prior distribution for \(\theta\). A noninformative prior is intended to represent prior ignorance and let the data speak for themselves.

We need these priors because they appear throughout the rest of the course:

  • As candidates for least favorable priors in minimax theory (Chapter 5).
  • As the natural priors in invariant decision problems (Chapter 6).
  • As the object of admissibility analysis for generalized Bayes rules (Chapter 8).

Two problems with the naive approach

The first idea: set \(\pi(\theta) = \text{constant}\). This fails immediately.

Problem 1: Parameterization dependence. Suppose \(\theta \in \mathbb{R}\) and \(\pi(\theta) = 1\). Reparameterize: \(\eta = e^\theta\). The induced prior on \(\eta\) is \[ \pi^*(\eta) = \pi(\log \eta) \cdot \left|\frac{d\theta}{d\eta}\right| = \frac{1}{\eta}, \] which is not constant. “Constant” depends on the parameterization.

Problem 2: Impropriety. On \(\mathbb{R}\), a constant function does not integrate to a finite value. It is not a proper probability density.

Location parameters: setup (Berger, Example 5)

Suppose \(\mathcal{X}= \Theta = \mathbb{R}^p\) and the density of \(X\) has the form \[ f(x \mid \theta) = f(x - \theta), \] depending on the data only through \(x - \theta\). Such a density is a location density, and \(\theta\) is a location parameter.

Standard examples:

  • \(\mathcal{N}(\theta, \sigma^2)\) with \(\sigma^2\) known
  • \(t(\alpha, \theta, \sigma^2)\) with \(\alpha, \sigma^2\) known
  • \(\text{Uniform}(\theta - 1, \theta + 1)\)

Location parameters: derivation via invariance

Instead of observing \(X\), observe \(Y = X + c\) for fixed \(c \in \mathbb{R}^p\). Define \(\mu = \theta + c\). Then \(Y\) has density \(f(y - \mu)\): structurally identical to the original.

If the problems are structurally identical, they should share the same noninformative prior. Writing \(\pi\) for both: \[ P_\pi(\theta \in A) = P_\pi(\theta \in A - c) \quad \text{for all } c \in \mathbb{R}^p. \] Any prior satisfying this is location invariant. In terms of densities: \[ \int_A \pi(\theta)\, d\theta = \int_A \pi(\theta - c)\, d\theta \quad \text{for all } A. \] This forces \(\pi(\theta) = \pi(\theta - c)\) for all \(\theta, c\), so \(\pi\) is constant.

The noninformative prior for a location parameter is \(\pi(\theta) = 1\).

Improper on \(\mathbb{R}^p\): \(\int_{\mathbb{R}^p} 1\, d\theta = \infty\).

The non-uniqueness issue

The argument required \(\pi = \pi^*\) exactly. But \(\pi\) and \(K\pi\) produce identical posteriors. Relaxing to proportionality: \[ \pi(\theta) = h(c)\, \pi(\theta - c) \] for some positive function \(h\). Setting \(\theta = c\): \(h(c) = \pi(c)/\pi(0)\), giving \[ \pi(\theta - c) = \frac{\pi(\theta)\, \pi(0)}{\pi(c)}. \]

Any such prior is relatively location invariant. Besides \(\pi(\theta) = 1\), this includes \(\pi(\theta) = \exp(\theta' z)\) for any fixed \(z\).

Selecting among them requires more structure. For certain problems, the natural choice is the right invariant Haar measure on the transformation group. We return to this in Chapter 6.

Scale parameters (Berger, Example 6)

A scale density has the form \(f(x \mid \sigma) = \sigma^{-1} g(x/\sigma)\), \(\sigma > 0\).

Standard examples: \(\mathcal{N}(0, \sigma^2)\); \(\text{Gamma}(\alpha, \beta)\) with \(\alpha\) fixed; \(\text{Exponential}(\sigma)\).

Derivation. Observe \(Y = cX\), \(c > 0\), define \(\eta = c\sigma\). Then \(Y\) has density \(\eta^{-1}g(y/\eta)\): structurally identical. The scale-invariance condition: \[ \pi(\sigma) = c^{-1}\pi(c^{-1}\sigma) \quad \text{for all } \sigma, c. \] Setting \(\sigma = c\): \(\pi(c) = c^{-1}\pi(1)\).

The noninformative prior for a scale parameter is \(\pi(\sigma) = \sigma^{-1}\).

Improper: \(\int_0^\infty \sigma^{-1}\, d\sigma = \infty\).

Benford’s law (Berger, Example 6 continued)

The distribution of first significant digits in tables of physical constants, population sizes, etc., follows Benford’s law: digit \(i\) occurs with relative frequency \(\log(1 + i^{-1})/\log 10\).

If table entries have a scale-invariant distribution \(\pi(u) = u^{-1}\), normalizing on \((1, 10)\): \[ P(\text{first digit} = i) = \int_i^{i+1} \frac{u^{-1}}{\log 10}\, du = \frac{\log(1 + i^{-1})}{\log 10}, \] matching the empirical observation. As \((a, b) \to (0, \infty)\), the probabilities converge to the same values regardless of \(a, b\).

Location-scale problems (Berger, Example 7)

Density: \(\sigma^{-1}f\bigl((x - \theta)/\sigma\bigr)\), both \(\theta \in \mathbb{R}\) and \(\sigma > 0\) unknown. Two approaches:

Approach 1 (Jeffreys’ rule). Fisher information for \(\mathcal{N}(\theta, \sigma^2)\): \[ \mathcal{I}(\theta, \sigma) = \begin{pmatrix} \sigma^{-2} & 0 \\ 0 & 2\sigma^{-2} \end{pmatrix}. \] Gives \(\pi(\theta, \sigma) = [\det \mathcal{I}(\theta, \sigma)]^{1/2} \propto \sigma^{-2}\).

Approach 2 (Independence). Multiply marginal noninformative priors: \(\pi(\theta, \sigma) = 1 \cdot \sigma^{-1} = \sigma^{-1}\).

These disagree. Jeffreys himself recommended \(\sigma^{-1}\), arguing that \([\det \mathcal{I}]^{1/2}\) sometimes gives inferior results in multiparameter settings. The prior \(\sigma^{-1}\) is also the right invariant Haar density.

Convention. For location-scale problems, use \(\pi(\theta, \sigma) = \sigma^{-1}\).

The Jeffreys prior (Berger $$3.3.3)

Definition: scalar case

For a general one-dimensional parameter \(\theta\), Jeffreys (1961) proposed \[ \pi(\theta) = [I(\theta)]^{1/2}, \] where \(I(\theta)\) is the expected Fisher information: \[ I(\theta) = -\mathbb{E}_{P_\theta}\!\left[\frac{\partial^2 \log f(X \mid \theta)}{\partial \theta^2}\right]. \]

Reparameterization invariance

Under \(\eta = h(\theta)\), Fisher information transforms as \(I_\eta(\eta) = I_\theta(\theta)/[h'(\theta)]^2\). The Jeffreys prior: \[ \pi_\eta(\eta) = [I_\eta(\eta)]^{1/2} = \frac{[I_\theta(\theta)]^{1/2}}{|h'(\theta)|}. \] The change-of-variables formula for densities gives the same expression: \(\pi_\eta(\eta) = \pi_\theta(\theta)/|h'(\theta)|\).

So the Jeffreys prior is invariant under reparameterization: compute it in one parameterization and transform, or compute directly in the new parameterization – the answer is the same.

This is the property that distinguishes it from the naive constant prior.

Definition: vector case

For \(\theta = (\theta_1, \ldots, \theta_p)'\): \[ \pi(\theta) = [\det \mathcal{I}(\theta)]^{1/2}, \] where \(\mathcal{I}(\theta)\) is the \(p \times p\) Fisher information matrix with \((i,j)\) entry \[ \mathcal{I}_{ij}(\theta) = -\mathbb{E}_{P_\theta}\!\left[\frac{\partial^2 \log f(X \mid \theta)}{\partial \theta_i\, \partial \theta_j}\right]. \]

Verification for location and scale

Location. If \(f(x \mid \theta) = f(x - \theta)\), then \(\frac{\partial^2}{\partial \theta^2}\log f(x \mid \theta)\) depends on \(x - \theta\). The expectation under \(P_\theta\) does not depend on \(\theta\) (substitute \(z = x - \theta\)). So \(I(\theta)\) is constant, and \(\pi(\theta) \propto 1\). Recovers the location-invariance result.

Scale. If \(f(x \mid \sigma) = \sigma^{-1}g(x/\sigma)\), a computation shows \(I(\sigma) \propto \sigma^{-2}\), giving \(\pi(\sigma) \propto \sigma^{-1}\). Recovers the scale-invariance result.

The Jeffreys prior unifies both cases under a single formula.

Board examples

Bernoulli. \(X \sim \text{Bernoulli}(\theta)\), \(\theta \in (0,1)\). \[ I(\theta) = \frac{1}{\theta} + \frac{1}{1-\theta} = [\theta(1-\theta)]^{-1}. \quad \Longrightarrow \quad \pi(\theta) = [\theta(1-\theta)]^{-1/2}. \] This is a \(\text{Beta}(1/2, 1/2)\) density. Proper.

Poisson. \(X \sim \text{Poisson}(\theta)\), \(\theta > 0\). \[ I(\theta) = \theta^{-1}. \quad \Longrightarrow \quad \pi(\theta) = \theta^{-1/2}. \quad \text{Improper.} \]

Normal mean. \(X \sim \mathcal{N}(\theta, \sigma^2)\), \(\sigma^2\) known. \[ I(\theta) = \sigma^{-2}. \quad \Longrightarrow \quad \pi(\theta) \propto 1. \quad \text{Improper.} \]

The multiplicity of noninformative priors (Berger, Example 8)

For the binomial parameter \(\theta \in [0,1]\), several noninformative priors:

Prior Source Formula
\(\pi_1(\theta) = 1\) Bayes, Laplace \(\text{Beta}(1,1)\)
\(\pi_2(\theta) = [\theta(1-\theta)]^{-1}\) Novick–Hall, Jaynes Improper
\(\pi_3(\theta) \propto [\theta(1-\theta)]^{-1/2}\) Jeffreys, Box–Tiao \(\text{Beta}(1/2, 1/2)\)

All three are defensible. For large \(n\), the posteriors are nearly identical. For small \(n\), they can differ – but for small \(n\) one typically has real prior information that should be used.

Criticisms and caveats (Berger $$3.3.4)

1. Violation of the Likelihood Principle. The Jeffreys prior depends on the expected Fisher information, which depends on the sampling model, not just the likelihood. Different experimental designs with the same likelihood can produce different Jeffreys priors.

2. The marginalization paradox (Dawid, Stone, and Zidek, 1973). In certain multiparameter problems, marginalizing under an improper prior can give results that depend on the order of integration.

3. Poor performance in high dimensions. The prior \(\pi(\theta) = 1\) for \(\theta \in \mathbb{R}^p\) implicitly assigns increasing weight to large \(\|\theta\|^2\) as \(p\) grows. The resulting generalized Bayes estimator \(\delta^0(x) = x\) is inadmissible for \(p \geq 3\) (James–Stein). Not an indictment of the Bayesian machinery; an indictment of the specific prior.

Convention (restricted parameter spaces). If \(\theta > 0\) in a location problem, the noninformative prior is still \(\pi(\theta) = 1\) on \((0, \infty)\). Inherited from the unrestricted space.

The posterior distribution (Berger $$4.1–4.2)

Why this section matters (Berger $$4.1)

The posterior distribution \(\pi(\theta \mid x)\) is the key computational object in Bayesian decision theory. Once we have it, choosing the optimal action reduces to a one-dimensional minimization (Section 4.4).

The posterior is not the goal; it is the intermediate step.

The mechanics of computing posteriors and conjugate families are review from 220A. The new content is the treatment of improper priors (Section 4.2.3), which connects to the noninformative priors we just derived.

Bayes’ theorem and conjugate families (Berger $$4.2.1–4.2.2)

For a proper prior \(\pi(\theta)\): \[ \pi(\theta \mid x) = \frac{f(x \mid \theta)\, \pi(\theta)}{m(x)}, \qquad m(x) = \int_\Theta f(x \mid \theta)\, \pi(\theta)\, d\theta. \]

A class \(\mathcal{G}\) of priors is conjugate for \(\mathcal{F}\) if the posterior stays in \(\mathcal{G}\).

Running example. \(X \sim \mathcal{N}(\theta, \sigma^2)\), prior \(\theta \sim \mathcal{N}(\mu, \tau^2)\). Posterior: \[ \pi(\theta \mid x) = \mathcal{N}\!\left(\frac{\tau^2}{\sigma^2 + \tau^2}\, x + \frac{\sigma^2}{\sigma^2 + \tau^2}\, \mu,\; \frac{\sigma^2 \tau^2}{\sigma^2 + \tau^2}\right). \]

Posterior mean: weighted average of \(x\) and \(\mu\), weights proportional to precisions. Posterior variance: smaller than both \(\sigma^2\) and \(\tau^2\).

Improper priors and formal posteriors (Berger $$4.2.3)

The mechanics carry over. If \(\pi(\theta)\) is improper, define \[ m(x) = \int_\Theta f(x \mid \theta)\, \pi(\theta)\, d\theta, \] and whenever \(0 < m(x) < \infty\): \[ \pi(\theta \mid x) = \frac{f(x \mid \theta)\, \pi(\theta)}{m(x)}. \]

This is the formal posterior. Not rigorously a conditional distribution, but justified heuristically: if \(\{\pi_n\}\) is a sequence of proper priors converging to \(\pi\), then \(\pi_n(\theta \mid x) \to \pi(\theta \mid x)\) under regularity conditions.

The computation is valid if and only if \(m(x)\) is finite and positive for (almost) all \(x\).

Normal location – it works

\(X \sim \mathcal{N}(\theta, \sigma^2)\), \(\pi(\theta) = 1\). \[ m(x) = \int_{-\infty}^\infty (2\pi\sigma^2)^{-1/2} \exp\!\left\{-\frac{(x - \theta)^2}{2\sigma^2}\right\} d\theta = 1. \] Posterior: \(\pi(\theta \mid x) = \mathcal{N}(x, \sigma^2)\). Proper.

This is the running example with \(\mathcal{N}(\mu, \tau^2)\) taken to its limit: as \(\tau^2 \to \infty\), the posterior mean \(\to x\), posterior variance \(\to \sigma^2\).

Normal scale – it works

\(X_1, \ldots, X_n \overset{\text{iid}}{\sim} \mathcal{N}(0, \sigma^2)\), \(\pi(\sigma) = \sigma^{-1}\), \(n \geq 1\). \[ m(x) = \int_0^\infty \sigma^{-n}(2\pi)^{-n/2} \exp\!\left\{-\frac{\sum x_i^2}{2\sigma^2}\right\}\sigma^{-1}\, d\sigma. \] Substitution \(u = 1/\sigma^2\): Gamma integral, finite when \(n \geq 1\) and \(\sum x_i^2 > 0\).

Posterior for \(\sigma^2\): Inverse-Gamma. Proper.

A case where it fails

\(X \sim \mathcal{N}(\theta, \theta^2)\) with \(\theta > 0\) (the mean equals the standard deviation). Prior \(\pi(\theta) = \theta^{-1}\). \[ m(x) = \int_0^\infty \frac{1}{\theta^2\sqrt{2\pi}} \exp\!\left\{-\frac{(x - \theta)^2}{2\theta^2}\right\} d\theta. \] The substitution \(u = x/\theta\) transforms this to an integral that diverges. The formal posterior is improper, and the Bayes computation is not valid.

This is a case where the “natural” noninformative prior fails.

Bayesian decision theory (Berger $$4.4)

Posterior expected loss (Berger, Definition 8)

The posterior expected loss of an action \(a\), when the posterior is \(\pi(\theta \mid x)\): \[ \rho(\pi(\theta \mid x),\, a) = \int_\Theta L(\theta, a)\, dF^{\pi(\theta \mid x)}(\theta). \]

A (posterior) Bayes action \(\delta^\pi(x)\) is any action \(a \in \mathcal{A}\) that minimizes \(\rho(\pi(\theta \mid x), a)\).

Normal form equals extensive form (Berger, Result 1)

Result 1. A Bayes rule \(\delta^\pi\) (minimizing \(r(\pi, \delta)\)) can be found by choosing, for each \(x\) with \(m(x) > 0\), an action minimizing the posterior expected loss.

Proof. By Fubini’s theorem (applicable since \(L(\theta, a) \geq -K > -\infty\)): \[ r(\pi, \delta) = \int_\Theta R(\theta, \delta)\, dF^\pi(\theta) = \int_{\{x:\, m(x)>0\}} \rho(\pi(\theta \mid x),\, \delta(x))\, dm(x). \] Minimize the integrand pointwise in \(x\). \(\square\)

The normal form (minimizing \(r(\pi, \delta)\) over all rules) and the extensive form (minimizing posterior expected loss for each \(x\)) give the same answer. We will virtually always work in the extensive form.

Generalized Bayes rules (Berger, Definition 9)

If \(\pi\) is an improper prior but \(\delta^\pi(x)\) minimizes the posterior expected loss for each \(x\) with \(m(x) > 0\), then \(\delta^\pi\) is called a generalized Bayes rule.

Generalized Bayes rules inherit the computational machinery of Bayesian decision theory. Their decision-theoretic properties (admissibility, minimaxity) are more delicate.

We study this in the next lecture (Section 4.8).

Squared-error loss: posterior mean (Berger, Result 3)

Result 3. If \(L(\theta, a) = (\theta - a)^2\), the Bayes rule is \[ \delta^\pi(x) = \mathbb{E}^{\pi(\theta \mid x)}[\theta]. \]

Proof. Differentiate \(\mathbb{E}^{\pi(\theta \mid x)}[(\theta - a)^2]\) with respect to \(a\): \[ -2\,\mathbb{E}^{\pi(\theta \mid x)}[\theta] + 2a = 0 \quad \Longrightarrow \quad a = \mathbb{E}^{\pi(\theta \mid x)}[\theta]. \quad \square \]

For quadratic loss \(L(\theta, a) = (\theta - a)'Q(\theta - a)\) with \(Q\) positive definite, the Bayes estimator is still the posterior mean. The matrix \(Q\) has no effect on the point estimate.

Running example under squared-error loss

\(X \sim \mathcal{N}(\theta, \sigma^2)\), prior \(\mathcal{N}(\mu, \tau^2)\). The posterior mean is \[ \delta^\pi(x) = \frac{\tau^2}{\sigma^2 + \tau^2}\, x + \frac{\sigma^2}{\sigma^2 + \tau^2}\, \mu. \] Weighted average, shrinking \(x\) toward \(\mu\). The shrinkage depends on \(\tau^2/\sigma^2\): the more spread the prior, the less shrinkage.

Under the noninformative prior \(\pi(\theta) = 1\): formal posterior is \(\mathcal{N}(x, \sigma^2)\), generalized Bayes estimator is \(\delta^0(x) = x\). The sample mean. The noninformative prior recovers it automatically.

This is Example 4 from Lecture 1 with \(c = \tau^2/(\sigma^2 + \tau^2)\): the Bayes rule under a normal prior gives the “optimal shrinkage” we could not determine there.

Absolute-error loss: posterior median (Berger, Result 5)

Result 5. If \(L(\theta, a) = |\theta - a|\), any median of \(\pi(\theta \mid x)\) is a Bayes estimator.

Proof. Let \(m\) be a median, \(a > m\). Then \[ L(\theta, m) - L(\theta, a) \leq (m-a)\,\mathbf{1}_{(-\infty, m]}(\theta) + (a-m)\,\mathbf{1}_{(m, \infty)}(\theta). \] Posterior expectation: \[ \leq (m-a)\cdot P(\theta \leq m \mid x) + (a-m)\cdot P(\theta > m \mid x) \leq 0 \] since \(P(\theta \leq m \mid x) \geq 1/2\). Symmetric argument for \(a < m\). \(\square\)

When the posterior is unimodal and symmetric, the posterior mean and median coincide. The Bayes estimator is the same under squared-error and absolute-error loss.

Linear loss: posterior fractile (Berger, Result 6)

Result 6. If \[ L(\theta, a) = \begin{cases} K_0(\theta - a) & \text{if } \theta \geq a \\ K_1(a - \theta) & \text{if } \theta < a, \end{cases} \] any \(\bigl(K_0/(K_0 + K_1)\bigr)\)-fractile of \(\pi(\theta \mid x)\) is a Bayes estimator.

\(K_0 = K_1\): the \(1/2\)-fractile is the median (recovering Result 5).

\(K_0 > K_1\) (underestimation is worse): Bayes estimator is above the median.

\(K_0 < K_1\) (overestimation is worse): Bayes estimator is below the median.

Hypothesis testing under 0–1 loss (Berger $$4.4.3)

Test \(H_0: \theta \in \Theta_0\) vs. \(H_1: \theta \in \Theta_1\). Actions: \(a_0\) (accept \(H_0\)), \(a_1\) (reject \(H_0\)).

Under 0–1 loss: \[ \mathbb{E}^{\pi(\theta \mid x)}[L(\theta, a_1)] = P(\Theta_0 \mid x), \qquad \mathbb{E}^{\pi(\theta \mid x)}[L(\theta, a_0)] = P(\Theta_1 \mid x). \]

Bayes action: accept the hypothesis with the larger posterior probability.

Under weighted 0–1 loss (\(K_0\) for Type I, \(K_1\) for Type II): accept \(H_0\) if \[ K_0\, P(\Theta_0 \mid x) \leq K_1\, P(\Theta_1 \mid x), \] equivalently if the posterior odds exceed \(K_0/K_1\).

Summary and preview

Summary

Noninformative priors (Berger $$3.3):

  • Location: \(\pi(\theta) = 1\). Scale: \(\pi(\sigma) = \sigma^{-1}\). Location-scale: \(\pi(\theta, \sigma) = \sigma^{-1}\).
  • General: Jeffreys prior \(\pi(\theta) = [I(\theta)]^{1/2}\), reparameterization invariant.
  • Improper, can fail (improper posteriors), but succeed in most standard problems.

Bayesian decision theory (Berger $$4.4):

  • Posterior expected loss \(\to\) minimize pointwise in \(x\) (extensive form = normal form).
  • Squared-error \(\to\) posterior mean. Absolute-error \(\to\) posterior median. 0–1 loss \(\to\) larger posterior probability.
  • Generalized Bayes rules: same machinery, but admissibility not guaranteed.
